What is Browser ?

*A browser is a software application used to access and view websites on the internet.

Examples :Google Chrome , Firefox , Microsoft Edge , Safari

What is Internet ?

*The Internet is a global network that connects millions of computers and devices, allowing them to share information and communicate with each other.

Examples : Watching videos on YouTube.

What is URL ?

*A URL (Uniform Resource Locator) is the address used to access a resource on the internet, like a website, image, or video.

Example :https://www.google.com – takes you to Google’s homepage.

What is HTTP ?

*HTTP is a set of rules that allows web browsers to request information from web servers, and receive web pages in response.

Example :http://news.ycombinator.com (not secure)

What is HTPPS ?

*HTTPS is the secure version of HTTP. It encrypts the data sent between your browser and the website, keeping it safe from hackers.

Example :https://www.google.com

What is Website and Webapp ?

*A website is a collection of web pages that provide information. It is mostly static (not interactive) and meant for reading or viewing content.

Example :www.wikipedia.org – Shows information about various topics.

*A web app is a website that is interactive and performs specific functions. It is usually dynamic and allows users to log in, input data, or perform actions.

Example :www.gmail.com – You can log in, send/receive emails.

What is Domain and division of Domain ?

*A domain is the main address of a website that people type into a browser to visit a site.

Example :www.amazon.in → amazon.in is the domain.

For the domain mail.google.com:
.com → Top-Level Domain (TLD)
google → Second-Level Domain
mail → Subdomain

What is IP Address ?

*An IP address (Internet Protocol Address) is a unique string of numbers that identifies a device on a network, like the internet. It works like a postal address, helping data find its way to the correct destination.

Example :192.168.1.1

What is the difference between Markup Language and Programming Language ?

Markup Language :

*A markup language is used to structure and format content on a webpage or document.It does not have logic

Programming Language :

*A programming language is used to create software applications that can perform tasks or solve problems.It contains logic
